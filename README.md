# unSQL Utility

A command line tool to run SQL on unstructured data. (Json and XML files)

## Getting Started


### Conditions

```
* Different 			
	- field <> value
* Equal 				
	- field = value
* Greater 			
	- field > value
* Greater or Equal 	
	- field >= value
* Less 				
	- field < value
* Less or Equal 		
	- field <= value
* In 				
	- field in (value1, value2)
* Not In 			
	- field notin (value1, value2)
* Like 				
	- field like value1
* Index 				
	- field index value1
```

### Aggregate Functions

```
* Average 	
	- avg(field)
* Count 		
	- count(field)
* Max 		
	- max(field)
* Min 		
	- min(field)
* Sum 		
	- sum(field)
```
